# Quick Start Guide

## Installation

```bash
npm install
```

## Run Tests

```bash
npm test
```

## Run Examples

```bash
npm run example
```

## Build

```bash
npm run build
```

## Project Files

- `src/merge.ts` - Main merge algorithm implementation
- `src/merge.test.ts` - Comprehensive unit tests (22 test cases)
- `src/example.ts` - Example usage demonstrations
- `src/index.ts` - Main export file

## Key Features

✅ Merges three sorted arrays without using any sort function
✅ O(n + m + p) time complexity
✅ Handles edge cases: empty arrays, duplicates, negative numbers
✅ 95.74% code coverage
✅ 22 comprehensive unit tests
✅ Full TypeScript support with type definitions
